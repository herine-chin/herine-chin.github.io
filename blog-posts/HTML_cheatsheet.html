HTML Cheatsheet

August 25, 2014

The Cheat Sheet.

When you are learning complex stuff like a programming language, it is great to have a cheat sheet or quick reference guide to basic or commonly used items. This will speed up your efficiency in writing and useful because we don't have eidetic memory.

Basic Structure

  • < html >

  • < head >

  • < title >website title< /title >

  • < /head >

  • < body >

  • content of website ...

  • < /body >

  • < /html >

Basics

  • < h? > ... < /h? > for headers

  • < p > ... < /p > for paragraphs

  • < b > ... < /b > to bold stuff

  • < i > ... < /i> to italicize stuff

  • < a href="url" > ... < /a > to reference links

  • < div > ... < /div > to divide page

  • < br > to add a break

  • < img src="url" alt="text" > for adding images

  • < ol > ... < /ol > for ordered lists

  • < ul > ... < /ul> for unordered lists
